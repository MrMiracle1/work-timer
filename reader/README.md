# Work Timer v1.4.0 读书功能使用指南

## 📖 功能概览

Work Timer v1.4.0 新增了完整的电子书阅读功能，让你在工作之余可以轻松享受阅读的乐趣。

## 🚀 快速开始

### 1. 进入读书模块

点击底部导航栏的"📖 读书"标签，即可进入读书模块。

### 2. 导入书籍

1. 点击顶部的"📖 导入书籍"按钮
2. 选择本地的电子书文件（支持 TXT、EPUB 格式）
3. 可以一次选择多个文件批量导入
4. 导入完成后，书籍会自动出现在书架上

### 3. 开始阅读

点击书架上的任意书籍卡片，即可进入阅读器开始阅读。

## 📚 书架功能

### 统计信息

书架顶部显示三个关键统计：
- **藏书**：书架上的总书籍数量
- **在读**：正在阅读中的书籍数量
- **读完**：已经读完的书籍数量

### 书籍卡片

每本书显示为一个精美的卡片，包含：
- 书籍封面（彩色渐变背景 + 📖 图标）
- 书名
- 章节数量
- 阅读进度（百分比 + 进度条）

### 删除书籍

1. 鼠标悬停在书籍卡片上
2. 点击右上角的"✕"按钮
3. 在确认弹窗中点击"删除"按钮
4. 书籍将从书架中移除（包括阅读进度）

## 📖 阅读器功能

### 基础操作

**翻页方式**：
- 点击底部的"上一页"/"下一页"按钮
- 使用键盘左右箭头键
- 拖动底部进度条快速跳转

**返回书架**：
- 点击顶部的"← 返回书架"按钮

### 章节导航

- 当前章节名称显示在底部中央
- 阅读进度（百分比）实时更新
- 进度条可拖动跳转到任意章节
- 首章/末章时翻页按钮自动禁用

### 文本标注

**高亮标记**：
1. 选中要标记的文本
2. 右键点击选中的文本
3. 在弹出菜单中选择"📝 高亮标记"
4. 文本将被高亮显示（黄色背景）

**复制文本**：
1. 选中要复制的文本
2. 右键点击选中的文本
3. 在弹出菜单中选择"📋 复制"
4. 文本已复制到剪贴板

## ⚙️ 阅读设置

点击阅读器顶部的"⚙️"按钮，打开阅读设置面板。

### 字体大小

四个档位可选：
- **小**：16px，适合大屏幕
- **中**：18px，默认设置，最舒适
- **大**：20px，适合远距离阅读
- **超大**：22px，适合视力较弱的用户

### 背景主题

三种主题可选：
- **默认**：白色背景 + 黑色文字，经典配色
- **护眼**：米白色背景 + 棕色文字，长时间阅读不疲劳
- **夜间**：黑色背景 + 浅灰文字，暗光环境下阅读

### 翻页模式

两种模式可选：
- **滚动**：传统滚动阅读，适合长篇内容
- **翻页**：分页显示，模拟纸质书阅读体验（预留功能）

### 设置特性

- 所有设置实时生效
- 设置自动保存到本地
- 下次打开应用时自动恢复设置

## 💾 进度保存

### 自动保存

- 每次切换章节时自动保存进度
- 保存当前阅读的章节索引
- 保存时间戳便于追踪

### 进度恢复

- 再次打开同一本书时
- 自动跳转到上次阅读的章节
- 无需手动查找阅读位置

### 数据存储

所有数据保存在浏览器本地存储（localStorage）中：
- 书籍列表
- 阅读设置
- 阅读进度
- 高亮标记

## 📝 支持的格式

### TXT 格式（完全支持）

- 自动识别常见章节标题格式：
  - "第一章"、"第1章"
  - "第一节"、"第1节"
  - "Chapter 1"、"CHAPTER 1"
  - "1."、"2."等数字开头
- 自动提取章节内容
- 支持中英文混排

### EPUB 格式（基础支持）

- 当前版本提供基础支持
- 可导入但解析功能有限
- 建议使用 TXT 格式获得最佳体验

## 🎯 使用技巧

1. **批量导入**：可以一次选择多个文件，快速建立书库
2. **键盘操作**：使用左右箭头键翻页，效率更高
3. **护眼模式**：长时间阅读建议使用护眼主题
4. **进度条**：想快速跳转到某一章，直接拖动进度条
5. **高亮标记**：重要内容及时标记，方便日后查找

## 🐛 常见问题

**Q: 导入的书籍没有显示章节？**
A: TXT 文件需要包含规范的章节标题（如"第一章"），否则会显示为单章。

**Q: 删除书籍后能恢复吗？**
A: 删除是永久性的，无法恢复。删除前请仔细确认。

**Q: 阅读进度会丢失吗？**
A: 只要不清除浏览器缓存，阅读进度会永久保存。

**Q: 支持同步到其他设备吗？**
A: 当前版本暂不支持，数据仅保存在本地浏览器中。

**Q: EPUB 格式为什么显示不正常？**
A: 当前版本对 EPUB 的支持有限，建议将书籍转换为 TXT 格式后导入。
