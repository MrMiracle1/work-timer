# AI自然语言指令控制使用指南

## 功能简介

Work Timer 现在支持通过自然语言指令控制各种功能，让你像和朋友聊天一样轻松操作！

## 快速开始

### 1. 配置API Key

首先需要在设置页面配置DeepSeek API Key：

1. 点击左上角菜单 → 设置
2. 找到"AI设置"区域
3. 输入你的DeepSeek API Key
4. 保存设置

### 2. 使用指令

在页面顶部的指令输入框中，用自然语言说出你的需求，点击"🤖 执行"按钮或按回车键即可。

## 使用技巧

### 1. 自然表达
不需要记忆固定命令，用你最习惯的方式说出需求即可。AI会智能理解你的意图。

**正确示例：**
- ✅ "我要打卡"
- ✅ "上班了"
- ✅ "打个卡吧"
- ✅ "开始工作啦"

### 2. 简洁明确
虽然支持自然语言，但表达越简洁明确，识别越准确。

**推荐：**
- ✅ "开始番茄钟"
- ✅ "查看时间轴"

**也支持：**
- ✅ "我想开始一个番茄钟专注工作"
- ✅ "能帮我看看今天的时间轴记录吗"

### 3. 快捷键
- 按 `Enter` 键快速提交指令
- 焦点在输入框时，输入框会自动放大，方便输入

### 4. 实时反馈
- 提交后会显示"⚙️ 执行中..."
- 执行成功会显示成功提示
- 执行失败会显示错误原因

## 常见问题

### Q: 提示"请先配置API Key"怎么办？
**A:** 需要先在设置页面配置DeepSeek API Key。点击提示后会自动跳转到设置页面。

### Q: AI识别不出我的指令怎么办？
**A:** 尝试使用更简洁明确的表达，或参考上面的示例指令。

### Q: 提示"当前不能执行"怎么办？
**A:** 某些操作有前置条件，例如：
- 上班打卡只能在未打卡时执行
- 下班打卡只能在已上班时执行
- 暂停番茄钟只能在番茄钟运行时执行

### Q: 可以同时执行多个指令吗？
**A:** 目前每次只能执行一个指令。如需执行多个操作，请分别输入。

## 技术原理

1. **意图识别**：调用DeepSeek大模型理解用户输入的自然语言
2. **指令解析**：将AI返回的JSON解析为操作类型和参数
3. **执行引擎**：根据操作类型调用对应的功能模块
4. **反馈机制**：实时显示执行状态和结果
