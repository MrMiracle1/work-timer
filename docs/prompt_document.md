# Work Timer Prompt指令文档

## 产品概述Prompt

```
你是一个专业的产品开发专家，请帮我创建一个名为"Work Timer"的工作时间管理工具。该工具主要用于帮助职场人士管理时间，通过倒计时显示距离下班、周末、发薪日等重要时间节点的时间，提升工作效率和时间管理能力。
```

## 核心功能逻辑Prompt

```
请实现以下核心功能模块：

1. 倒计时模块
   - 下班倒计时：实时显示距离下班时间的剩余时间，精确到秒
   - 周末倒计时：显示本周剩余工作时间（小时和分钟）
   - 发薪日倒计时：根据用户设置显示距离发薪日的天数
   - 期待时间倒计时：智能显示下一个重要时间点（午饭时间或最近假期）
   - 其他假期倒计时：显示所有法定节假日倒计时

2. 工资进度模块
   - 进度条显示：可视化展示本月工资获取进度
   - 进度文本：显示具体进度百分比数值

3. 日历模块
   - 月份切换：支持前后月份切换查看
   - 日期类型标识：工作日、周末、公休假期、串休上班日、自定义假期
   - 日期交互：点击日期可切换其类型
   - 图例说明：清晰展示各种日期类型的颜色标识

4. 设置模块
   - 工作时间设置：上班时间和下班时间设置
   - 发薪日设置：发薪日类型选择和具体日期设置
   - 开发者设置：自定义时间模式和缓存清除功能
   - 自定义事件：支持添加和管理自定义倒计时事件

5. 首次设置模块
   - 初始配置弹窗：首次使用时引导用户进行基本设置
```

## 法定节假日Prompt

```
请预设以下法定节假日：

1. 固定公历节日
   - 元旦：1月1日
   - 劳动节：5月1日
   - 国庆节：10月1日

2. 农历节日（近似公历日期）
   - 春节：2月10日（2024年）
   - 端午节：6月10日（2024年）
   - 中秋节：9月17日（2024年）

3. 特殊节日
   - 清明节：4月4日或5日（根据年份调整）
   - 补休日：根据节假日是否在周末自动计算
```

