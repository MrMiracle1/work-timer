# Work Timer 用户手册

## 产品简介

Work Timer 是一款专为职场人士设计的工作与假期管理工具。它帮助您清晰掌握距离下一个假期、发薪日或重要事件的时间，合理规划工作进度，提升时间管理效率。产品支持桌面与移动端，适合个人使用。

## 快速开始

### 1. 首次使用设置

首次打开 Work Timer 时，系统会弹出设置窗口，您需要进行以下设置：

1. **设置工作时间**
   - 上班时间：设置您的上班时间（默认为09:00）
   - 下班时间：设置您的下班时间（默认为17:30）

2. **设置发薪日**
   - 发薪日类型：选择"固定日期"或"每月最后一天"
   - 每月发薪日：如果选择固定日期，请设置具体的发薪日（1-31日）

3. **保存设置**
   - 点击"保存"按钮完成首次设置

### 2. 界面导航

Work Timer 提供三种主要功能模块，您可以通过标签页进行切换：

- **倒计时**：查看各种重要时间节点的倒计时
- **日历**：查看月份日历和日期类型
- **设置**：修改个人设置和添加自定义事件

在桌面端，标签页位于页面顶部；在移动端，标签页位于页面底部。

## 功能详解

### 1. 倒计时模块

倒计时模块是 Work Timer 的核心功能，为您提供多种时间节点的倒计时信息。

#### 1.1 下班倒计时
显示距离今天下班时间的剩余时间，精确到秒。点击卡片会显示鼓励提示。

#### 1.2 周末倒计时
显示本周剩余工作时间，以小时和分钟为单位展示。

#### 1.3 发薪日倒计时
显示距离下一个发薪日的天数，根据您在设置中配置的发薪日类型和日期计算。

#### 1.4 期待时间倒计时
智能显示下一个重要时间点，可能是午饭时间或最近的假期。

#### 1.5 其他假期倒计时
显示所有法定节假日的倒计时，按时间顺序排列。

### 2. 工资进度模块

在倒计时页面的下半部分，显示本月工资获取进度：

- **进度条**：可视化展示从上月发薪日到本月发薪日的进度
- **进度文本**：显示具体进度百分比数值

### 3. 日历模块

日历模块帮助您查看和管理日期类型：

#### 3.1 查看日历
- 点击"◀"和"▶"按钮可以切换月份
- 不同类型的日期以不同颜色标识：
  - 绿色：工作日
  - 蓝色：周末
  - 红色：公休假期
  - 黄色：串休上班日
  - 绿色（浅）：自定义假期

#### 3.2 修改日期类型
- 点击任意日期可以切换其类型
- 工作日可以切换为公休假期
- 周末可以切换为串休上班日

#### 3.3 图例说明
页面底部提供图例说明，帮助您识别不同颜色代表的日期类型。

### 4. 设置模块

设置模块允许您个性化配置 Work Timer：

#### 4.1 用户设置
- **上班时间**：修改您的上班时间
- **下班时间**：修改您的下班时间
- **发薪日类型**：选择"固定日期"或"每月最后一天"
- **每月发薪日**：如果选择固定日期，请设置具体的发薪日

修改设置后，点击"更新"按钮保存更改。

#### 4.2 开发者设置
- **开启自定义时间**：启用后可以设置自定义时间用于测试
- **自定义时间**：设置自定义时间（需要先开启自定义时间模式）
- **删除缓存**：清除所有设置并恢复为默认设置

#### 4.3 自定义事件
您可以添加自己的倒计时事件：

1. **事件名称**：输入事件的名称
2. **事件时间**：选择事件的具体时间
3. **重复类型**：选择事件的重复方式（不重复/每天/每周/每月/每年）
4. **添加事件**：点击"添加"按钮保存事件

添加的自定义事件会显示在倒计时页面的"其他假期"区域。

## 使用技巧

### 1. 移动端使用
- 在移动端，底部导航栏便于单手操作
- 点击倒计时卡片可以获得鼓励提示
- 日历中的日期点击区域较大，便于触屏操作

### 2. 数据管理
- 所有设置和自定义事件都保存在您的浏览器本地存储中
- 更换浏览器或清除浏览器数据会导致设置丢失
- 建议定期记录重要的自定义事件以防丢失

### 3. 时间设置建议
- 根据实际工作时间设置上下班时间，以获得准确的倒计时
- 根据公司实际发薪日设置发薪日类型和日期
- 合理设置自定义事件，避免添加过多不重要的事件

## 常见问题

### 1. 倒计时显示不准确
**问题**：倒计时时间与实际时间不符
**解决方案**：
- 检查设备时间是否准确
- 检查是否开启了开发者模式的自定义时间
- 刷新页面重新加载

### 2. 设置无法保存
**问题**：修改设置后刷新页面发现设置未保存
**解决方案**：
- 确保点击了"保存"或"更新"按钮
- 检查浏览器是否禁用了localStorage
- 尝试清除缓存后重新设置

### 3. 日历日期类型无法修改
**问题**：点击日期无法切换类型
**解决方案**：
- 确保点击的是日期数字区域
- 检查是否有JavaScript错误
- 刷新页面重新尝试

### 4. 自定义事件不显示
**问题**：添加的自定义事件没有在倒计时页面显示
**解决方案**：
- 检查事件时间是否设置正确
- 确认事件是否已成功添加（可在设置页面查看事件列表）
- 刷新页面重新加载

## 数据隐私说明

Work Timer 非常重视您的隐私：

1. **本地存储**：所有数据都保存在您的浏览器本地，不会上传到任何服务器
2. **无数据收集**：我们不会收集任何用户使用数据或个人信息
3. **数据控制**：您可以随时通过"删除缓存"功能清除所有个人数据

## 技术支持

如果您在使用过程中遇到任何问题，可以通过以下方式获取帮助：

1. **查看文档**：参考本用户手册和产品文档

# 节假日配置与日历使用指南

## 导入/重置/导出节假日
- 打开“设置”页的“节假日设置”。
- 在“法定节假日JSON”与“周末上班JSON”中粘贴或编辑 JSON。
- 点击“导入节假日”即时应用；“重置为默认”恢复到内置 2026 集合；“导出配置JSON”下载合并后的配置文件。

## 查看节假日显示
- 日历中的公休假期以红色标签显示节日名称，并以背景色高亮。
- 周末上班日以黄色背景显示。
- 可点击日期在“正常周末/周末上班”和“正常工作日/公休假期”之间切换，修改会写入浏览器本地存储。

## 月份切换
- 在“日历”页使用“◀ / ▶”按钮切换上/下月。
- PC 端已修复按钮事件绑定问题，切换更稳定。

## 倒计时卡片提示
- 鼠标悬停在“下班倒计时/本周剩余工作时间/发薪倒计时”卡片上，会显示计算依据与当前参数。
