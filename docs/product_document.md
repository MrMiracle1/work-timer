# Work Timer 产品文档

## 产品概述

Work Timer 是一款专为职场人士设计的工作与假期管理工具。它帮助用户清晰掌握距离下一个假期、发薪日或重要事件的时间，合理规划工作进度，提升时间管理效率。产品支持桌面与移动端，适合个人和团队使用。

## 产品定位

Work Timer 定位于提升职场人士时间管理效率的工具型产品，通过可视化倒计时和进度追踪，帮助用户更好地规划工作与休息时间。

## 核心功能模块

### 1. 倒计时模块

#### 1.1 下班倒计时
- 实时显示距离下班时间的剩余时间
- 精确到秒的倒计时显示
- 点击卡片可显示鼓励提示

#### 1.2 周末倒计时
- 显示本周剩余工作时间
- 以小时和分钟为单位展示
- 包含周末日期信息

#### 1.3 发薪日倒计时
- 根据用户设置显示距离发薪日的天数
- 支持固定日期和月末最后一天两种模式
- 显示发薪日具体日期

#### 1.4 期待时间倒计时
- 智能显示下一个重要时间点（午饭时间或最近假期）
- 根据当前时间自动切换显示内容
- 提供更近的目标激励

#### 1.5 其他假期倒计时
- 显示所有法定节假日倒计时
- 按时间顺序排列展示
- 包含节日名称和具体日期

### 2. 人民日报阅读模块

#### 2.1 文章列表
- 显示当天所有人民日报文章标题
- 带序号标记的列表展示
- 支持悬停高亮效果

#### 2.2 文章阅读
- 点击文章标题可阅读完整内容
- 模态框展示文章正文
- 支持滚动查看长文

#### 2.3 日期选择
- 支持选择任意日期
- 查看历史报纸内容
- 默认显示当天报纸

#### 2.4 刷新功能
- 支持手动刷新获取最新内容
- 显示加载进度提示
- 错误提示和异常处理

### 3. 工资进度模块

#### 2.1 进度条显示
- 可视化展示本月工资获取进度
- 从上月发薪日到本月发薪日为完整周期
- 实时更新进度百分比

#### 2.2 进度文本
- 显示具体进度百分比数值
- 提供直观的进度描述

### 4. 日历模块

#### 3.1 月份切换
- 支持前后月份切换查看
- 显示当前月份标题

#### 3.2 日期类型标识
- 工作日（绿色）
- 周末（蓝色）
- 公休假期（红色）
- 串休上班日（黄色）
- 自定义假期（绿色）

#### 3.3 日期交互
- 点击日期可切换其类型
- 工作日可切换为公休假期
- 周末可切换为串休上班日

#### 3.4 图例说明
- 清晰展示各种日期类型的颜色标识
- 帮助用户快速识别不同日期类型

### 5. 设置模块

#### 4.1 工作时间设置
- 上班时间设置（时:分格式）
- 下班时间设置（时:分格式）

#### 4.2 发薪日设置
- 发薪日类型选择（固定日期/月末最后一天）
- 固定发薪日设置（1-31日）

#### 4.3 开发者设置
- 自定义时间模式开关
- 自定义时间设置（年-月-日 时:分格式）
- 缓存清除功能

#### 4.4 自定义事件
- 事件名称输入
- 事件时间设置
- 重复类型选择（不重复/每天/每周/每月/每年）
- 事件添加与删除功能

### 6. 首次设置模块

#### 5.1 初始配置
- 上班时间默认设置（09:00）
- 下班时间默认设置（17:30）
- 发薪日类型默认设置（固定日期）
- 发薪日默认设置（每月1号）

## 数据存储

### 1. 本地存储
- 使用localStorage存储用户设置
- 保存自定义事件数据
- 保存假期类型配置

### 2. 存储字段说明
- workStartTime: 上班时间
- lunchStartTime: 午休开始时间
- lunchEndTime: 午休结束时间
- workEndTime: 下班时间
- salaryType: 发薪日类型
- salaryDay: 固定发薪日
- customEvents: 自定义事件
- holidays: 自定义假期
- workdaysOff: 公休假期
- weekendsWork: 串休上班日
- developerMode: 开发者模式开关
- customTime: 自定义时间
- deepseekApiKey: DeepSeek API 密钥
- lastClockInType: 上次打卡类型
