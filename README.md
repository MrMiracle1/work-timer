# Work Timer 产品说明书

> 版本：v1.4.1 | 更新日期：2024-12-12

## 产品定位

Work Timer 是一款专为职场人士设计的工作与假期管理工具。它帮助用户清晰掌握距离下一个假期、发薪日或重要事件的时间，合理规划工作进度，提升时间管理效率。产品支持桌面与移动端，适合个人和团队使用。

---

## 核心功能

### 1. 假期与事件倒计时

- 首页展示距离下一个假期、发薪日或自定义事件的剩余时间。
- 支持多种事件类型（如法定假期、周末、工作日、发薪日、自定义事件）。
- 以卡片形式直观展示，便于用户快速了解时间节点。

### 2. 日历视图

- 提供完整的月份日历，支持左右切换月份。
- 日历中不同类型的日期（如假期、周末、工作日等）以不同颜色高亮显示。
- 图例说明帮助用户快速识别日期类型。
- 左右分栏布局：左侧日历，右侧上方添加事件，下方查看事件列表。

### 3. 工作进度追踪

- 展示本周、本月等周期的工作进度。
- 进度条动画效果，直观反映已完成比例。
- 支持自定义周期和目标设置。

### 4. 🎰 摸鱼吉日签系统（v1.3.3 新增）

- **AI智能生成**：上班打卡时生成趣味十足的摸鱼吉日签，替代传统留言。
- **三大核心信息**：
  - 🌟 **今日摸鱻运势**：用星级或趣味文案描述运势
  - ⏰ **最佳摸鱻时间**：AI推荐最适合摸鱻的时间段及原因
  - 💡 **摸鱻锦囊**：2-3条实用且有趣的摸鱻建议
- **再求一签功能**：不满意可以重新生成，直到遇到心仪的签文。
- **Header展示**：首页顶部直接显示今日摸鱻运势，时刻提醒。
- **积极幽默风格**：所有内容都充满正能量和幽默感，让打卡变得有趣。

### 5. 💰 工资计算与收入展示（v1.3.3 新增）

- **月工资设置**：在首次设置向导和设置页面中配置每月工资。
- **智能日薪计算**：自动按当月天数计算每日平均工资（月工资 ÷ 当月天数）。
- **收入展示**：下班打卡时明确告知“今日赚了XXX元”，收入信息高亮显示。
- **AI幽默总结**：工作总结由AI生成，语言风趣幽默，包含所有关键信息。

### 6. ✨ 打卡特效系统（v1.3.3 新增）

- **阳光特效（上班打卡）**：50个黄色光点从顶部缓缓飘落，带有光晕效果，营造温暖阳光氛围。
- **礼花特效（下班打卡）**：3组彩色粒子从中心向外爆炸，带有重力效果，寄意圆满完成工作。
- **高性能实现**：使用HTML5 Canvas绘制，requestAnimationFrame优化动画流畅度。

### 7. 📖 读书功能（v1.4.0 新增，v1.4.1 增强）

- **书架管理**：统计藏书、在读、读完书籍数量，书籍卡片展示。
- **本地导入**：支持从本地导入TXT、EPUB等格式的电子书文件。
- **智能解析**：自动识别章节标题，提取目录结构。
- **阅读器**：
  - 章节翻页：上下章节快速切换
  - 进度条：拖动快速跳转到任意章节
  - 键盘快捷键：左右箭头翻页
- **阅读设置**：
  - 字体大小：小/中/大/超大 4档调节
  - 主题颜色：默认/护眼/夜间 3种主题
  - 翻页模式：滚动/翻页模式切换
- **🎯 沉浸式阅读模式**（v1.4.1 新增）：
  - 专注模式按钮：一键开启/关闭专注模式
  - 自动隐藏工具栏：鼠标移到顶部时显示
  - 淡化辅助元素：进度条和导航栏透明度降低
  - 状态持久化：刷新页面后保持专注模式
- **✨ 文本标注增强**（v1.4.1 新增）：
  - 多色高亮：黄色、绿色、蓝色、粉色、紫色 5种颜色
  - 颜色选择：右键菜单显示颜色选择器
  - 高亮笔记：为高亮文本添加个人注释
  - 笔记图标：带笔记的高亮显示📝图标
  - 高亮管理：查看所有高亮和笔记，支持筛选和跳转
  - 复制文本：快捷复制选中内容
- **🤖 AI问书功能**（v1.4.1 新增）：
  - 智能提问：选中文本右键“问AI”
  - 上下文提取：自动获取前后段落作为语境
  - AI解释：使用DeepSeek进行语义理解
  - 继续提问：支持追问和深入讨论
  - 保存笔记：将AI解释保存为笔记
  - 问答记录：自动保存所有问答历史
- **进度保存**：自动保存阅读进度，下次打开继续阅读。
- **书籍删除**：二次确认弹窗，防止误删。

### 8. 设置与个性化

- 用户可自定义工作时间、假期类型、提醒方式等。
- 支持添加自定义事件，满足个性化需求。
- 支持月工资配置，自动计算日薪。

### 8. 友好提示与反馈

- 操作成功或异常时，顶部弹出提示，提升交互体验。
- 空状态时显示友好文案，避免用户迷失。

### 9. 响应式设计

- 自动适配桌面和移动端，保证不同设备上的使用体验。
- 移动端底部导航栏，便于单手操作。

## 典型使用场景

- **个人时间管理**：随时掌握距离下一个假期或发薪日的时间，合理安排工作与休息。
- **团队协作**：统一查看公司假期安排，提升团队沟通效率。
- **自定义提醒**：为重要项目、会议等设置倒计时，避免遗忘。

## 用户价值

- 提升时间管理能力，减少遗忘和焦虑。
- 让假期、发薪日等重要节点一目了然，增强工作动力。
- 支持个性化设置，满足不同用户需求。

## 产品结构概览

- 首页：倒计时卡片、进度条、主要事件展示。
- 日历页：月份日历、日期类型高亮、图例说明。
- 读书页：书架管理、阅读器、阅读设置。
- 设置页：工作时间、假期、自定义事件管理。
- 移动端底部导航：快速切换各功能模块。
